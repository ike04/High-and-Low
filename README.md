# High-and-Low
トランプを使った「ハイ&ロー」ゲームを作成しました。

### ルール
1.コンピューターカードを表向きに1枚出します。

2.コンピュータがもう一枚裏向きでカードを出します。

<img src="https://user-images.githubusercontent.com/48178913/105621138-d4c06680-5e47-11eb-875b-542a754ddea1.jpg" width=15%>

3.プレーヤーは裏向きのカードが表向きのカードよりも大きいか小さいかをあてます。
4.当たったらプレーヤに１ポイント、はずれたらコンピュータに１ポイント加算されます。

<img src="https://user-images.githubusercontent.com/48178913/105621194-5dd79d80-5e48-11eb-8f61-6c7b80ddbc0a.jpg" width=15%>
